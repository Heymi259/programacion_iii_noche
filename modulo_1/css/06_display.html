DISPOSICION RELATIVA

Un texto es un conjunto coherente y ordenado de enunciados, orales o escritos, que forma una unidad de comunicación con un propósito específico y un sentido completo, utilizando un sistema de signos (como un idioma) para transmitir un mensaje en un contexto dado. La palabra "texto" proviene del latín textus, que significa "tejido", refiriéndose a la forma en que los diferentes elementos se entrelazan para formar un todo con significado.
